== Description ==

WP Pay Per View is a powerful wordpress plugin that gives WordPress website administrators the ability to offer streaming video immediately to their customers once successful payment is made. This plugin easily allows Divi users to add WP Pay Per View videos to their Divi website as a Divi module.


== Installation ==

1. Upload the plugin files to the /wp-content/plugins/wp-pay-per-view directory, or install the plugin through the WordPress plugins screen directly.

2. Activate the plugin through the \'Plugins\' screen in WordPress


== Frequently Asked Questions ==

Q. Do I need to purchase the WP Pay Per View plugin to use the WP Pay Per View Divi plugin?

A. Yes, you can purchase the WP Pay Per View plugin at https://wppayperview.com/pricing/


Q. Can I use this on websites that are not developed with the Divi builder? 

A. No, this plugin only works with sites using the Divi builder.
